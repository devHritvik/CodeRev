CSS Selectors

Basic Selectors

Different ways to select elements in CSS.

/* Element Selector */
p { color: blue; }

/* Class Selector */
.highlight { background: yellow; }

/* ID Selector */
#main-title { font-size: 24px; }

/* Attribute Selector */
[type="text"] { border: 1px solid #ccc; }

This is a paragraph with element selector

Combinators

Combining selectors for more specific targeting.

/* Descendant Selector */
div p { color: green; }

/* Child Selector */
ul > li { border-left: 2px solid red; }

/* Adjacent Sibling */
h2 + p { font-style: italic; }

/* General Sibling */
h2 ~ p { color: purple; }

CSS3 Animations & Transitions

Transitions

Create smooth transitions between property values.

.transition-example {
  transition: all 0.3s ease;
  background: #3498db;
  padding: 20px;
  color: white;
}

.transition-example:hover {
  background: #2ecc71;
  transform: scale(1.1);
}
Hover me to see transition

Keyframe Animations

Create complex animations using keyframes.

@keyframes bounce {
  0% { transform: translateY(0); }
  50% { transform: translateY(-20px); }
  100% { transform: translateY(0); }
}

.animation-example {
  animation: bounce 1s infinite;
  background: #e74c3c;
  padding: 20px;
  color: white;
}
