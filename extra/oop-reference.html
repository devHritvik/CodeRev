Classes & Objects

Class Definition

Creating classes and objects in different languages.

Java public class Person {
  private String name;
  private int age;

  public Person(String name, int age) {
    this.name = name;
    this.age = age;
  }

  public void introduce() {
    System.out.println("Hi, I'm " + name);
  }
}

Python class Person:
  def __init__(self, name, age):
    self.name = name
    self.age = age

  def introduce(self):
    print(f"Hi, I'm {self.name}")

Key concepts:

  • Class as a blueprint
  • Instance variables
  • Constructor method
  • Instance methods

Encapsulation

Access Modifiers

Controlling access to class members.

Java public class BankAccount {
  private double balance;
  protected String accountNumber;
  public String ownerName;

  public void deposit(double amount) {
    if (amount > 0) {
      balance += amount;
    }
  }

  public double getBalance() {
    return balance;
  }
}

Access levels:

  • private: Class only
  • protected: Class and subclasses
  • public: Anywhere
  • default: Package only

Inheritance

Class Inheritance

Creating class hierarchies.

Java class Animal {
  protected String name;

  public void makeSound() {
    System.out.println("Some sound");
  }
}

class Dog extends Animal {
  public void makeSound() {
    System.out.println("Woof!");
  }

  public void fetch() {
    System.out.println("Fetching...");
  }
}

Inheritance concepts:

  • Base/Parent class
  • Derived/Child class
  • Method overriding
  • Additional functionality

Polymorphism

Method Overriding

Different implementations of the same method.

Java class Shape {
  public double getArea() {
    return 0;
  }
}

class Circle extends Shape {
  private double radius;

  public double getArea() {
    return Math.PI * radius * radius;
  }
}

class Rectangle extends Shape {
  private double width, height;

  public double getArea() {
    return width * height;
  }
}

Polymorphism types:

  • Compile-time (Method overloading)
  • Runtime (Method overriding)
  • Interface-based
  • Abstract class-based

Abstraction

Abstract Classes

Creating abstract classes and methods.

Java abstract class Vehicle {
  protected String brand;

  public abstract void start();
  public abstract void stop();

  public void displayBrand() {
    System.out.println("Brand: " + brand);
  }
}

class Car extends Vehicle {
  public void start() {
    System.out.println("Car starting...");
  }

  public void stop() {
    System.out.println("Car stopping...");
  }
}

Abstraction concepts:

  • Abstract classes
  • Abstract methods
  • Concrete methods
  • Implementation in subclasses

Interfaces

Interface Definition

Creating and implementing interfaces.

Java interface Drawable {
  void draw();
  void resize(int percentage);
}

class Circle implements Drawable {
  public void draw() {
    System.out.println("Drawing circle");
  }

  public void resize(int percentage) {
    System.out.println("Resizing circle by " + percentage + "%");
  }
}

Interface concepts:

  • Contract definition
  • Multiple implementation
  • Method signatures
  • Default methods
